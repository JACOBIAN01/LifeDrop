# 🩸 LifeDrop – Community + Institutional Blood Donation Platform  

LifeDrop is a **next-gen blood donation platform** that connects **donors, patients, hospitals, and NGOs** in a single network. Unlike traditional apps that either focus on *only community-based donations* or *only hospitals/blood banks*, LifeDrop bridges both worlds — making blood donation **faster, safer, and more reliable**.  

---

## 🚀 Why LifeDrop?  
Blood donation apps exist, but they often suffer from:  
- ❌ No real-time donor availability  
- ❌ Privacy risks (sharing phone numbers publicly)  
- ❌ No gamification or donor engagement  
- ❌ Weak hospital integration  
- ❌ Lack of insights into demand/supply  

✅ **LifeDrop solves these gaps with a simple, scalable tech stack.**  

---

## ✨ Features (MVP)  

- 🔹 **Community Blood Donation Network** – Users can post requests; donors get notified and can show interest.  
- 🔹 **Hospital & NGO Accounts** – Hospitals/NGOs can register, post verified requests, and get matched with donors.  
- 🔹 **Privacy-Safe Matching** – Donors & requesters connect via in-app system (no need to expose phone numbers).  
- 🔹 **Gamification** – Donors earn badges like *Life Saver* 🏆 to encourage repeated donations.  
- 🔹 **Geo-based Matching** – Nearest donors get priority via location-based search.  
- 🔹 **Simple Dashboard** – Donors, Patients, and Hospitals each get role-based dashboards.  

---

## 🛠️ Tech Stack  

**Frontend:**  
- React (with Hooks + Router)  
- Tailwind CSS (for clean, modern UI)  

**Backend:**  
- Node.js + Express (API layer)  

**Database & Auth:**  
- Firebase Firestore (NoSQL database)  
- Firebase Auth (secure authentication & role management)  
- Firebase Storage (upload verification docs)  

**Other Services:**  
- Google Maps API (location + routing)  
- Firebase Cloud Messaging (notifications)
